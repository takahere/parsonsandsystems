# Parsons and Systems Website

A minimalist website for Parsons and Systems, based in Tokyo.

## Features

- Clean, modern design
- Responsive layout that works on all devices
- Click counter that tracks user interactions
- Clickable email link

## Setup

Simply host these files on your web server with the domain parsonsandsystems.com:

- `index.html` - Main HTML structure
- `styles.css` - Styling for the website
- `script.js` - JavaScript functionality for the click counter and email link

## Customization

To update the email address, modify the `mailto:` link in `script.js`:

```js
window.location.href = 'mailto:your-actual-email@parsonsandsystems.com';
```
